13. Библиотеки CSS для сложных анимаций. Основы работы с Bootstrap.

В проекте Hlegals сверстать блок Що ми вміємо робити краще за інших? (скриншот во вложениях), используя классы из Bootstrap. Контейнер можно расширить, прописав отдельный класс на него, чтобы соответствовало макету. Необходимо убедиться в правильности работы верстки в адаптиве.

акже вспомним одну из первых лекций - подключение шрифта не через Гугл. Во вложении к домашнему заданию вы найдете архив со шрифтами, которые используются в макете. Их надо подключить на сайт, желательно сконвертировать в актуальные форматы (woff, ttf).
