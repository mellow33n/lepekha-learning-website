03. Основы CSS и работа со шрифтами

Стилизуем формы, которые вы делали в прошлый раз. Нужно использовать эти свойства:
Форма 1
1) background-color
2) color
3) box-shadow
4) border
5) display: inline-block
6) vertical-align

Форма 2
1) background-color
2) color
3) border
4) font-weight
5) border-radius
6) vertical-align

Форма 3
1) color
2) background-color
3) column-count

В последней форме не делаем картинку в селекте, обрезанный бордер (границу) в верхней части у инпутов, не стилизуем чекбоксы, Не обязательно делать красивую кнопку для кнопки загрузки файла.

Задание достаточно творческое. Задавайте вопросы в чат, если будут вопросы по свойствам. Всем успехов!
